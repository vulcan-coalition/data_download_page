Data Information

The dataset is organized into two main groups within the zip file. The main 'group' contains data with superior voice quality, while the 'secondary_group' features recordings of lower quality, which may be useful for handling noisy data, low volume, or voices with dialects. Each folder in the dataset includes a transcription in a CSV file, named identically to the folder, and the audio recordings are provided in .flac format.

File Format

The CSV provided contains two column headers: 'filepath' and 'instruction'. The 'filepath' column lists the names of the files, indicating the location of each voice recording within the dataset. The 'instruction' column details the specific directives and spoken content given by the data provider to each individual who recorded the voice files.Each CSV file are encripted with UTF-8 and the instruction are in Thai.

Main Group :

The main group contains higher quality voice data and includes a total of 498,175 files.

Secondary Group :

The 'secondary group' contains lower quality voice data, useful for processing challenges such as background noise or dialects, and has a total of 141,667 files.

Dataset part

To facilitate ease of access and accommodate various bandwidth capabilities, the dataset has been segmented into subfolders and is available for download in three different sizes: Large, Medium, and Small.

Large Tab:

- Intended for users with a robust internet connection.
- Each zip file contains 100,000 files from either the main group (total of 498,175 files) or the secondary group (total of 141,667 files).
- Accompanying each batch is a CSV file with data corresponding to these files, bearing the same name as the zip file.
- The size of each Large zip file is approximately 21,000 MB.

Medium Tab:

- Each zip file contains 10,000 files, catering to users with moderate download capacity.
- Like the Large tab, each Medium zip file is paired with a CSV containing data for its respective files and shares the same naming convention.
- The size of each Medium zip file is about 2,100 MB.

Small Tab:

- Ideal for preliminary testing or users with unstable internet connections.
- Each zip file comprises 1,000 files, following the structure of the Medium and Large tabs.
- The size of each Small zip file is around 210 MB.

Please note:

- Combining the first 10 files from the smaller size group will yield an equivalent dataset to the first file of the next larger size group.
- Downloading all the zip files within a chosen size category will provide the complete dataset, identical to downloading the full dataset in one go.

Choose the tab that best suits your needs to begin downloading the subfolders of the dataset.
