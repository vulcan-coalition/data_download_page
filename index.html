Smart Home Thai language TTS dataset

This open Thai language speech collection, is a product of the Vulcan Coalition.
This dataset is designed specifically for voice command recognition in smart home devices. It comprises a comprehensive collection of voice recordings that are used to facilitate the development and improvement of voice-activated control systems within a smart home environment. These recordings include a variety of commands that one would typically use to interact with smart home technology, such as turning lights on and off, or adjusting the thermostat.
The project receives sponsorship from the Thailand Digital Economy Promotion Agency.
Each audio file in the dataset has been either transcribed or recorded by individuals with disabilities. As a result, there may be variations in quality due to the differing recording devices used.

Data Information

The dataset is organized into two main groups within the zip file. The main group contains data with superior voice quality, while the secondary group features recordings of lower quality, which may be useful for handling noisy data, low volume, or voices with dialects. Each folder in the dataset includes a transcription in a CSV file, named identically to the folder, and the audio recordings are provided in .flac format.

The provided CSV features two columns: 'filepath' and 'instruction'. The 'filepath' column enumerates the file names, designating the location of each voice recording in the dataset. The 'instruction' column outlines the precise instructions and spoken content issued by the data provider to the individuals recording the files. Each CSV file is encoded in UTF-8, and the instructions are provided in Thai.

Main Group :

The main group contains higher quality voice data and includes a total of 498,175 files.

Secondary Group :

The secondary group contains lower quality voice data, useful for processing challenges such as background noise or dialects, and has a total of 141,667 files.

Partial Download

To facilitate ease of access and accommodate various bandwidth capabilities, the dataset has been segmented into subfolders and is available for download in three different sizes: Large, Medium, and Small.

Please note:

- Combining the first 10 files from the smaller size group will yield an equivalent dataset to the first file of the next larger size group.
- Downloading all the zip files within a chosen size category will provide the complete dataset, identical to downloading the full dataset in one go.

Choose the tab that best suits your needs to begin downloading the subfolders of the dataset.

Small :

- Small zip files are set up with 1,000 files, ideal for initial testing.
- Main group files are labeled sgroup_{file_number}.zip, e.g., sgroup_1.zip.
- Secondary group files are named s2group_{file_number}.zip, e.g., s2group_1.zip.
- Each has a size of about 210 MB.

Medium :

- Medium zip files hold 10,000 files each, with a matching CSV.
- Main group files are named mgroup_{file_number}.zip, e.g., mgroup_1.zip.
- Secondary group files follow the pattern m2group_{file_number}.zip, e.g., m2group_1.zip.
- The size for each is approximately 2,100 MB.

Large :

- Each Large zip file contains 100,000 files, with an accompanying CSV file named identically to the zip.
- Zip files for the main group are named Lgroup_{file_number}.zip, e.g., Lgroup_1.zip.
- For the secondary group, the naming is L2group_{file_number}.zip, e.g., L2group_1.zip.
- The size of each Large zip file is approximately 21,000 MB.
