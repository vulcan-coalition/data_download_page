Thai language STT dataset

This open Thai language speech collection, is a product of the Vulcan Coalition.
This dataset is a comprehensive Thai Speech-to-Text collection meticulously transcribed by individuals with disabilities, ensuring a unique perspective in the transcription process.
It has undergone a basic quality checking procedure to maintain a standard level of accuracy and reliability.
The project receives sponsorship from the Thailand Digital Economy Promotion Agency.

Data Information

The dataset comprises 28,771 audio files, each accompanied by labeled text provided in a .csv file format for easy access and analysis. The audio data is categorized into three group.Each folder in the dataset includes the audio recordings provided in .mp3 format.

The provided CSV features two columns: 'filepath' and 'labels'. The 'filepath' column enumerates the file names and subfolder name, designating the location of each voice recording in the dataset. The 'labels' column outlines the precise instructions and spoken content issued by the data provider to the individuals recording the files. Each CSV file is encoded in UTF-8, and the instructions are provided in Thai.
